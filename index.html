manipulable.css 디자이너 (목업)

각 선택자별 배경색·배경이미지·외곽선·radius·글자색을 조절하면 미리보기와 오른쪽 CSS가 실시간으로 갱신됩니다.

⚠ 미리보기와의 실시간 연동에 실패했습니다. 브라우저가 로컬 파일 간 접근을 차단하고 있을 수 있습니다 (로컬 서버로 열면 해결됩니다).

① 다이스/액션 아이콘 폰트 (:before)

dicefont, pictos 아이콘 종류는 해당 링크에 잘 정리되어 있습니다. (별개로 커스텀 시트 제작법을 공부하기 좋은 사이트입니다.)

※ 실제 폰트 목록 확정 전까지는 목업 옵션만 제공됩니다.

※ 주사위/액션 버튼 안에 표시되는 글자입니다. dicefont 등 아이콘 폰트를 쓰는 경우 해당 폰트의 코드 문자를 입력하세요.

② 시트 배경 (.charsheet)

캐릭터 시트 전체의 배경/테두리/기본 글자색입니다.

※ 실제 폰트 목록 확정 전까지는 목업 옵션만 제공됩니다.
